Стек технологий, с которым мы научились работать

Vue

Vue — это прогрессивный фреймворк для создания пользовательских интерфейсов.

NodeJS

Программная платформа, превращающая JavaScript из узкоспециализированного языка в язык общего назначения.

PostgreSQL

Свободная объектно-реляционная система управления базами данных.

Express.js

Фреймворк для Node.js, который реализовывает слой функций, необходимых для создания эффективных приложений и API.

Sequelize

ORM-библиотека для приложений на Node.js, которая осуществляет сопоставление таблиц в бд и отношений между ними с классами.

Figma

Онлайн-сервис для разработки интерфейсов и прототипирования.

GraphQL

Фреймворк для Node.js, который реализовывает слой функций, необходимых для создания эффективных приложений и API.

JWT

ORM-библиотека для приложений на Node.js, которая осуществляет сопоставление таблиц в бд и отношений между ними с классами.

Apollo

Онлайн-сервис для разработки интерфейсов и прототипирования.

Webpack

JavaScript сборщик модулей с открытым исходным кодом.

Git

Распределённая система управления версиями.

Chart.js

JavaScript-библиотека визуализации данных.
